Thực hiện theo các bước sau đây để cài đặt
1. Cài JDK 11+ (nếu chưa cài)
2. Cài SQL Server 2008+ (nếu chưa cài)
3. Chạy QLCuaHangText.sql để tạo CSDL AHTKStore
4. Chạy Setup.exe để cài ứng dụng AHTKStore
5. Chạy và đăng nhập với tài khoản (username: mã nhân viên,
password: số điện thoại)
Ví dụ: 
5.1 Quản lí:
+ username: TCLKT2
+ password: 0977019702
5.2 Nhân viên bình thường
+ username: TCLKT1
+ password: 0339657164
